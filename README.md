# E-Commerce Site Project Structure

This directory contains the complete structure for an E-Commerce Site project.

## Root Directory Structure:

- **frontend/**  
  Contains the React application code.
  - **public/**  
  - **src/**  
  - **package.json**  
  - **README.md**  

- **backend/**  
  Contains the Node.js/Express server code.
  - **controllers/**  
  - **models/**  
  - **routes/**  
  - **config/**  
  - **package.json**  
  - **server.js**  

- **database/**  
  Contains database schema and migration files.
  - **migrations/**  
  - **models/**  
  - **seeds/**  

- **.gitignore**
- **README.md**  
- **package.json**  
- **docker-compose.yml**  
- **.env**
